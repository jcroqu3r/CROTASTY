# Fonts for deployment

To keep the exact same typography on GitHub/GitHub Pages, add the licensed PP Editorial Old files in this folder with these names:

- PP-Editorial-Old-Ultralight.woff2
- PP-Editorial-Old-Ultralight.woff

The site already loads:

- DM Sans from Google Fonts
- PP Editorial Old from ./fonts/ via @font-face

If only one format is available, keep the `.woff2` file at minimum.
